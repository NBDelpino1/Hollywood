#### Hollywood Trivia - a simple javaScript game 

##### User has 30 seconds to answer each question about Hollywood celebs.

##### Kick off - user presses start button to start the game
![image](https://cloud.githubusercontent.com/assets/18251657/24129923/1c56308c-0dbc-11e7-83fe-2699c6f271b4.png)

##### User is presented with a question and has 30 seconds to select 1 of 4 answers presented
![image](https://cloud.githubusercontent.com/assets/18251657/24129973/61dabc0e-0dbc-11e7-9352-88b90f3bb809.png)

##### User can either get it correct 
![image](https://cloud.githubusercontent.com/assets/18251657/24129992/79d7db3e-0dbc-11e7-9037-c465582ceaf3.png)

##### Get it wrong
![image](https://cloud.githubusercontent.com/assets/18251657/24130006/8d60a870-0dbc-11e7-9376-b3372fe0b9f1.png)

##### Or run out of time
![image](https://cloud.githubusercontent.com/assets/18251657/24130031/a7f062a2-0dbc-11e7-9012-d56674f4e616.png)

##### After the user has answered all of the questions OR if time has run out, they are presented with the results screen
##### The user will then have the option of re-starting the game by pressing the restart button
![image](https://cloud.githubusercontent.com/assets/18251657/24130073/eb781c90-0dbc-11e7-97b4-895afa0ae1de.png)
